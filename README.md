# 无人机地面控制站 (UAV GCS)

基于 Web 的无人机地面控制站系统，包含地图定位、障碍物与航线规划、飞行监控、通信链路展示四大模块。

## 功能模块

### 3.1 地图定位模块
- **3.1.1 WGS-84与GCJ-02坐标转换**
  - 支持 WGS-84 ↔ GCJ-02 双向转换
  - 支持 BD-09 坐标系转换
  - 完整的单元测试页面：`test_coord.html`
  
- **3.1.2 OpenStreetMap地图显示**
  - 集成 OpenStreetMap 开源地图
  - 校内起始点设置（默认北京地区坐标：116.3147, 39.9897）
  - 支持地图缩放、平移、中心点显示

### 3.2 障碍物与航线规划模块
- **3.2.1 多边形圈选障碍物**
  - 点击地图添加顶点，绘制任意多边形障碍物
  - 双击或"完成绘制"按钮结束绘制
  
- **3.2.2 障碍物高度设置与JSON保存**
  - 自定义障碍物名称和高度
  - 支持导出为 JSON 文件
  - 支持从 JSON 文件导入障碍物数据
  - 示例数据：`data/sample_obstacles.json`

- **3.2.3 飞行高度/安全半径设置**
  - 自定义飞行高度
  - 自定义安全半径
  - 支持飞越/绕飞两种策略切换

- **3.2.4 飞越/绕飞路径规划**
  - 飞越策略：提升飞行高度越过障碍物
  - 绕飞策略：规划绕行航点避开障碍物
  - 显示路径长度和航点信息

### 3.3 飞行监控模块
- 实时飞行状态显示（飞行模式、GPS状态、卫星数量）
- 姿态数据可视化（横滚、俯仰、航向）
- 电池状态监控（电压、电流、剩余电量、预估飞行时间）
- 位置与高度信息
- 实时位置地图追踪
- 飞行日志记录

### 3.4 通信链路展示模块
- **3.4.1 GCS-OBC-FCU拓扑图**
  - SVG 绘制的三层网络拓扑图
  - GCS（地面控制站）→ OBC（机载计算机）→ FCU（飞行控制器）
  - 实时链路状态显示（延迟时间）

- **3.4.2 MAVLink数据流与报文显示**
  - 模拟 MAVLink 报文流
  - 支持报文类型过滤
  - 显示统计信息（总包数、丢包率、接收速率、数据量）
  - 支持调节显示速率（慢/正常/快）
  - 支持暂停/继续/清空

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- [Leaflet.js](https://leafletjs.com/) - 开源地图库
- [OpenStreetMap](https://www.openstreetmap.org/) - 地图瓦片服务

## 本地运行

直接用浏览器打开 `index.html` 即可，或启动一个本地 HTTP 服务器：

```bash
# 方式一：Python
python -m http.server 8080

# 方式二：Node.js (需要 http-server)
npx http-server -p 8080
```

然后访问 `http://localhost:8080`

## 部署到 GitHub Pages

1. 在 GitHub 上创建新仓库
2. 将所有文件推送到仓库的 `main` 分支
3. 在仓库 Settings → Pages 中：
   - Source 选择 `Deploy from a branch`
   - Branch 选择 `main`，目录选择 `/ (root)`
4. 等待几分钟后访问 `https://<username>.github.io/<repo-name>/`

## 使用说明

### 地图定位模块
1. 在左侧输入 WGS-84 经纬度坐标
2. 点击转换按钮查看 GCJ-02 坐标
3. 设置校内起始点坐标，点击"在地图上标记"

### 障碍物与航线规划
1. 点击"绘制多边形"按钮
2. 在地图上点击添加顶点
3. 点击"完成绘制"或双击结束
4. 设置障碍物名称和高度，点击"保存障碍物"
5. 设置起点和终点（点击按钮后在地图上选点）
6. 选择规划策略（飞越/绕飞）
7. 点击"开始路径规划"查看结果

### 飞行监控
- 查看实时飞行数据（模拟数据自动更新）
- 查看飞行轨迹和日志

### 通信链路
- 查看 GCS-OBC-FCU 拓扑图和链路状态
- 监控 MAVLink 报文流
- 使用过滤和速度控制功能

## 坐标系统说明

- **WGS-84**: World Geodetic System 1984，GPS 使用的坐标系
- **GCJ-02**: 国测局坐标系，火星坐标系，中国大陆地图使用
- **BD-09**: 百度坐标系，百度地图使用
